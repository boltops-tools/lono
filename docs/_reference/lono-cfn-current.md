## Usage

    lono cfn current

## Description

Current stack that you're working with.


## Options

```
[--rm], [--no-rm]            # Remove all current settings. Removes `.lono/current`
[--name=NAME]                # Current stack name.
[--verbose], [--no-verbose]  
[--noop], [--no-noop]        
```
